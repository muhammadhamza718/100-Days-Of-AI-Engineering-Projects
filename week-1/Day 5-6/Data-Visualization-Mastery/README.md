# Data Visualization Mastery

This project implements a comprehensive data visualization curriculum featuring a Titanic Exploratory Data Analysis (EDA) mini-project using Python with matplotlib, seaborn, and plotly libraries.

## Getting Started

1. Install dependencies: `uv pip install -r requirements.txt` or use the included libraries
2. Run the visualization gallery: `uv run python visualization_gallery.py`
3. Run the Titanic EDA: `uv run python Titanic_EDA.py`

## Structure

- `visualization_gallery.py`: Examples from matplotlib, seaborn, and plotly libraries
- `Titanic_EDA.py`: Comprehensive EDA with 15+ plots following 7-step methodology
- `insights.md`: Top 5 findings from the analysis
